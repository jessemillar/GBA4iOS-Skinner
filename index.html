Drag and drop a .png or .json file to get started

When finished, click "Export" and save the displayed text in a file named "info.json"

Valid images and image sizes are as follows:
    iPhone Portrait
        "portrait.png"
        640x480
    iPad Portrait
        "portrait-ipad.png"
        768x427
    Retina iPad Portrait
        "portrait-ipad@2x.png"
        1536x854
    iPhone Landscape
        "landscape.png"
        960x640
    iPhone Widescreen Landscape
        "landscape@R4.png"
        1136x640
    iPad Landscape
        "landscape-ipad.png"
        1024x768
    Retina iPad Landscape
        "landscape-ipad@2x.png"
        2048x1536
